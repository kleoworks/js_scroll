What are Databases?
A structured set of data

Databases are a collection of information, they are used for creating, manipulating, and deleteing data. There are many different stypes of databases and they can be classifed in a basic approach of relational database or a non-relational database. Relationshiops means a set of our data is related to another section of our data. Meanwhile in a non-relational database there is not a common related setion of our data. Our datbases are put into sections of data called schemas, insdie each schema each data set is oraganized into tables, those tables contain columns and rows to house that particular set of data.
